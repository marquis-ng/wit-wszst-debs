wit-wszst-debs

Debian repository for Wiimms ISO Tools and Wiimms SZS Tools.

Introduction

This repository provides Debian packages of Wiimms ISO Tools and Wiimms SZS Tools. It checks for a new version of them every 24 hours. If a new version is found, it will be compiled and uploaded here as a Debian package.

Installation

Debian repository

sudo wget https://marquis-ng.github.io/wit-wszst-debs/wit-wszst-debs.list -O /etc/apt/sources.list.d/wit-wszst-debs.list
          wget -O- https://marquis-ng.github.io/wit-wszst-debs/KEY.gpg | gpg --dearmor | sudo tee /etc/apt/trusted.gpg.d/wit-wszst-debs.gpg > /dev/null
          sudo apt update

Package

Name Summary Command
wit Wiimms ISO Tools 
sudo apt install wit
wszst Wiimms SZS Tools 
sudo apt install wszst

After that, the utility will be installed!
